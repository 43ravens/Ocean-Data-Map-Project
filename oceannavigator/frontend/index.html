Ocean Navigator (Beta)

  1. What is the ocean navigator

    The ocean navigator is a user-friendly online map of scientific ocean data of ocean models. It includes a number of different variables such as temperature and ocean currents. The tool is developed by scientists but is designed for range of end users. Anyone can use it, whether they are in the scientific community, industry, academia or general public.

  2. Objective of Portal

    The Ocean Navigator prototype portal is being designed with the goal to enable the discovery and accessibility of ocean data from ocean prediction systems. Operational prediction systems from Canada and other global systems are available via this tool. Several other sources of data are available, including, validation data sources and limited observational data. More information on the data available can be found in the Ocean Navigator. For help in using the Ocean Navigator please see our user guide.

    Please note that this site is still in active development and may have some bugs or experience other issues in rendering data, for access to raw data please see our THREDDS server. To report a bug please e-mail: oceandatamap@gmail.com

  3. Recommended Use of Data

    These datasets are not intended for navigational purposes. It is expected that users of this portal will have an interest in oceanographic data and the intention is to provide oceanographic context for a wide range of users.

  4. Data Producer Responsibility

    The developers are not responsible for the use made of the data made accessible via this portal, or errors or omissions that potentially may occur in the data sets. While we aim to make accessible the most recent and up to date data, occasionally a delay in the data feed to the portal may be experienced.

  5. Identification of Data Sources

    Use of the downloaded data should identify the source of the data. Plots downloaded from the portal will have a reference to the data source.

    Suggested citation: Dataset, http://navigator.oceansdata.ca (year)
    Example: GIOPS daily averages, http://navigator.oceansdata.ca, 2017

    If you would like further information about how to reference the data sources that the Ocean Navigator uses please e-mail DFO.OceanNavigator-NavigateurOcean.MPO@dfo-mpo.gc.ca
